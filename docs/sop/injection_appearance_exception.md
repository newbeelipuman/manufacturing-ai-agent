# 注塑件外观不良 SOP

适用范围：注塑件出现划伤、黑点、色差、毛边、缩水、油污、变形等外观异常时的现场处理。

## 处理步骤

1. 隔离异常批次，暂停将该批次流入下一工序或客户订单。
2. 通知质检进行复判，记录不良现象、不良数量、批次、SKU、仓库和关联工单。
3. 生产主管确认是否存在模具、参数、原料或操作异常。
4. 质检根据标准判定返工、报废、让步接收或继续观察。
5. 如果异常批次已经关联客户订单，应同步销售、仓库和生产主管评估交付风险。
6. Agent 只提供查询和建议，不自动执行出库、调账、审批、下单等业务写操作。

## 人工确认

涉及放行、报废、返工、让步接收、客户通知或库存调整时，必须由业务人员在企业系统中人工确认。
