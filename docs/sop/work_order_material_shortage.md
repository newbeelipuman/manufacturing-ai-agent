# 工单缺料与齐套处理 SOP

适用范围：工单开工前发现物料库存不足、齐套不完整、领料不足或采购到货不确定时的处理。

## 处理步骤

1. 生产主管确认工单号、成品 SKU、计划数量、计划开工日期和工单状态。
2. 仓库逐项复核 BOM 物料的可用库存、锁定库存、批次和仓库位置。
3. 对存在缺口的物料，采购确认在途采购单、预计到货日期和可替代供应方案。
4. 生产主管评估是否延期工单、调整排产顺序、降低计划数量或等待齐套。
5. 如果缺料影响客户订单交付，应同步销售评估交期风险。
6. Agent 只提供齐套分析和建议，不自动开工、领料、调账、排产或审批。

## 人工确认

涉及开工、领料、排产调整、物料替代或客户交期沟通时，必须由业务人员在企业系统中人工确认。
